Drone simulation

This is a drone simulation coded in java and javafx using an object-oriented structure. Different types of drones are put on a canvas and navigate obstacles. An interface with different buttons allow the user to create their own simulation, save it and load a previously saved simulation. Here is a list of the different entities in the simulation:

-Normal Drone: A drone that spawns randomly in the drone arena and moves in a random direction at a random speed. The drone bounces off the edges of the arena and other entities.
-Boulder: An obstacle which spawns randomly on the arena.
-Zombie Drone: A drone which will chase other drone entities and also turn them into zombie drones on contact. They are programmed to hunt the closest drone entity, and cannot chase drones behind boulders.
-Player Drone: A drone which is controlled by the player using WASD.
-Laser: A laser which will shoot from the left side to the right of the screen at the height of the user's mouse click on the arena. Each entity has health, and if the laser makes contact with an entity, the entity will lose 1 health. When an entity is out of health, it disappears from the arena. The laser can also be shot at a random height using a button.

The project repository can be found on my github here: https://github.com/MehdiSouameur/DroneSimulation
